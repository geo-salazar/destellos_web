Notas:
1. Cambia el número de WhatsApp 50688888888 por el número real de la tienda.
2. Puedes reemplazar las imágenes SVG por fotos reales manteniendo los mismos nombres o cambiando las rutas en el HTML.
3. El carrito funciona con JavaScript y guarda los productos en localStorage del navegador.
4. Este proyecto puede abrirse directamente en Dreamweaver.
